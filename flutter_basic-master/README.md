## 변경사항
- 204페이지 6.3.2 StatefulWidget 클래스 동작

집필 당시(1.12.13버전)에는 책에서 설명한 것 처럼 다음 화면을 표시할 때 이전 페이지도 다시 그려진다고 되어 있으나, 현재는 이 동작은 변경되었습니다. Stateless와 동일하게 동작하여 나중에 그리는 페이지만 그려집니다.
