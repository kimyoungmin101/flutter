# 설명

인스타그램 스토리 형식처럼 내가 올린 사진을 클릭하였을 경우 넘어갈 수 있고 그렇게 스토리 형식처럼 자동으로 넘길 수 도 있습니다.
내가 원하는 사진이라면 이미지를 Pick할 수 있는 기능도 구현하였습니다.

# 프로젝트 목표
Test를 위한 미리 앱을 만들어달라는 회사의 요청으로 제작을 진행하였습니다. 나만의 이미지 를 Pick하는 기능을 구현하였고 이미지를 스와이핑 하는 동작은 인스타그램 스토리 형식을 착안하였습니다.

# 프로젝트 설명
인스타그램 스토리 형식처럼 내가 올린 사진을 클릭하여 넘길 수 있고 다른 주제로 넘어가 스토리 형식 이미지뷰를 관찰이 가능하도록 설정하여 내가 원하는 사진을 선택하는 기능을 구현하였습니다.

# 개발 방법
dart를 이용 플러터 클라이언트 개발을 진행하였고 db작업은 firebase의 firebasestorage를 이용하여 이미즐 저장하여 스토리 형식 이미지 파일을 넘길 수 있게 해주었고 database는 모두 firebase에 담아 활용하여 클라이언트에서 이미지들을 불러 만들어 진행하였습니다.
